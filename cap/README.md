# CAP — Constitutional Authority Protocol

**Draft:** [draft-sato-soos-cap-03](https://datatracker.ietf.org/doc/draft-sato-soos-cap/)
**Status:** Active — IETF Datatracker
**Vienna tier:** Tier 1 (Vienna critical — due June 14)
**Web page:** https://soosproject.ai/drafts/cap

## What this draft specifies

CAP defines the constitutional prohibition layer of the SOOS governance stack. It specifies the four-tier authority model (Tier 0-A absolute prohibitions through Tier 3 resource constraints), the Cedar policy enforcement architecture, the Constitutional Mandate Registry (CMR) protocol, and the Prohibition Clearance Record (PCR) mechanism for Tier 0-B exceptions.
