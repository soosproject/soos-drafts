# CAP-RRS — Constitutional Authority Protocol Regulation Record Schema

**Draft:** [draft-sato-soos-cap-rrs-00](https://datatracker.ietf.org/doc/draft-sato-soos-cap-rrs/)
**Status:** Active — IETF Datatracker
**Vienna tier:** Tier 3 (Launch complete — due June 28)
**Web page:** https://soosproject.ai/drafts/cap-rrs

## What this draft specifies

CAP-RRS specifies the schema for Regulation Records — structured JSON representations of legal compliance obligations — and the Cedar Compilation Profile that translates them into Cedar policies for GEC enforcement. It also specifies the Constitutional Mandate Registry (CMR) distribution protocol, the certification chain, and the versioned update model.

Compliance with applicable law should be a package import, not a Cedar authoring problem.
