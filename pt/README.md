# PT — Progressive Trust

**Draft:** [draft-sato-soos-pt-01](https://datatracker.ietf.org/doc/draft-sato-soos-pt/)
**Status:** Active — IETF Datatracker
**Vienna tier:** Tier 3 (Launch complete — due June 28)
**Web page:** https://soosproject.ai/drafts/pt

## What this draft specifies

PT defines the behavioral trust model for AI agents, specifying how an agent's track record of governed execution earns expanded operational authority over time. It specifies the trust score model, the Progressive Trust Record format, behavioral credit history accumulation, and the trust level to authority expansion mapping.

- **GAR** — PT scores are computed from the GAR audit trail
- **AEP** — PT level determines the authority headroom available at AEP PLAN step
